AI Powered Landing Page Generator
Overview

An AI-driven web application that generates professional, responsive landing pages dynamically using the Gemini API. The system processes user input prompts and converts structured AI responses into production-ready HTML templates.

Tech Stack

PHP (Backend Logic)

Gemini API (AI Content Generation)

HTML5

CSS3

Features

Dynamic landing page generation from user prompts

Direct integration with Gemini API

Backend response parsing and structured content rendering

Modular and reusable frontend components

Responsive design for multiple screen sizes

How It Works

User enters a prompt.

Backend sends structured request to Gemini API.

API response is parsed and formatted.

Dynamic HTML template is generated and rendered.

Setup Instructions

Clone the repository

Add your Gemini API key in configuration file

Run using local PHP server

Access via browser
